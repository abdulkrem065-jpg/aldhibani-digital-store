# Qaroni AI Engine — Task Lifecycle Specification
## مواصفة دورة حياة المهام وتتبع الحالات الـ 12 للمحرك

---

## 1. الحالات الـ 12 لدورة حياة المهمة (The 12 Distinct States)

لتفادي الانتقالات العشوائية والقفز التشغيلي، تمر أي تذكرة ترحيل أو تعديل بـ 12 حالة محصورة ومتسلسلة:

1. **`Created` (تم الإنشاء):** التذكرة مسجلة في النظام ويجري تدوين مبررها الدستوري.
2. **`Analyzing` (قيد التحليل):** الوكيل `Qaroni_Analyzer` يقوم بقراءة ومقارنة المخططات واحتساب الفجوات الهيكلية.
3. **`Planning` (قيد التخطيط):** صياغة مسودة التعديل ومخطط الـ SQL والـ TypeScript المقترح.
4. **`Validating` (قيد التحقق الفني):** إطلاق محاكاة الـ Sandbox وتشغيل الفحوصات المعرفية والـ Linter.
5. **`WaitingApproval` (بانتظار التفويض البشري):** اكتمال المحاكاة بنجاح تام، وتوقف المحرك تلقائياً بانتظار توقيع المالك الرقمي (OTP).
6. **`Approved` (تم التفويض):** استلام ومطابقة توقيع المالك الرقمي ونقله لمرحلة التنفيذ.
7. **`Deploying` (قيد البث الحي):** تفعيل بوابة الكتابة وبدء ترحيل السيرفر كمعاملة واحدة (Single Transaction).
8. **`Verifying` (قيد التحقق البعدي):** تشغيل اختبارات الاستقرار (Smoke Test) والتحقق من سياسات RLS على السيرفر الحي.
9. **`Completed` (اكتمل بنجاح):** اجتياز كافة الاختبارات البعدية، وتصفير الكاش، وتنظيف البيئات الموقتة، وتدوين تقرير التدقيق.
10. **`RollbackInitiated` (بدء التراجع):** رصد خطأ تقني أو فشل في الفحص البعدي، والبدء بإعادة النظام للحالة الصفرية المستقرة.
11. **`RolledBack` (تم التراجع والإنقاذ):** تراجع قاعدة البيانات والأكواد للحالة السابقة بنجاح، وتجميد المهمة وإخطار المالك.
12. **`Failed` (فشل نهائي):** تعذر الإصلاح التلقائي أو تعطل روتينات التراجع (يتطلب معالجة بشرية فورية).

## 2. جدول شروط الانتقال والتحولات الآمنة (State Transition Matrix)

| الحالة الحالية (Current State) | الحالة المستهدفة (Target State) | شروط الانتقال الأمن (Transition Criteria) | الوكيل المسؤول (Agent) |
| :--- | :--- | :--- | :--- |
| `Created` | `Analyzing` | قراءة التكوين والتحقق الدستوري المبدئي | `Qaroni_Reader` |
| `Analyzing` | `Planning` | احتساب الفروقات الفنية وصياغة كشف الفجوات بنجاح | `Qaroni_Analyzer` |
| `Planning` | `Validating` | صياغة مسودات التعديل Up/Down بالكامل | `Qaroni_MigrationBuilder` |
| `Validating` | `WaitingApproval` | نجاح محاكاة الـ Sandbox وتماسك العلاقات بنسبة 100% | `Qaroni_Validator` |
| `WaitingApproval` | `Approved` | استلام ومطابقة توقيع المالك ورمز OTP المشفر | `State Machine Controller` |
| `Approved` | `Deploying` | تفعيل بوابة الكتابة الآمنة `WriteGateway` | `Qaroni_Executor` |
| `Deploying` | `Verifying` | تطبيق الترحيلات سحابياً بنجاح وتحديث كاش الـ API | `Qaroni_Executor` |
| `Verifying` | `Completed` | نجاح اختبارات الدخول وسياسات RLS على الإنتاج | `Qaroni_Auditor` |
| `Verifying` | `RollbackInitiated` | فشل أي فحص بعدي أو رصد زيادة في استثناءات الأخطاء | `Qaroni_Auditor` |
| `RollbackInitiated` | `RolledBack` | نجاح تطبيق تراجع قاعدة البيانات والأكواد السابقة | `Qaroni_Auditor` |
