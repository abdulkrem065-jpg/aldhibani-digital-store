* **عدد الأحداث المبرمجة داخل الكتالوج (Total Catalog Events):** `9` أحداث متكاملة مدفوعة بالعمليات.
* **عدد حالات دورة الحياة للمهام (Task Lifecycle States):** `12` حالة متميزة وشروط انتقال صارمة.

## 2. تفصيل الهياكل والمجلدات الـ 17 الفرعية (Directory Breakdown)

4. `constitution/` — ميثاق الدستور الأعلى والسيادة البشرية.
5. `database/` — مجلد استبقاء مخططات الاستعلامات المحلية (Placeholder).
6. `deployment/` — مجلد استبقاء أكواد وتحديث الكاش سحابياً (Placeholder).
7. `docs/` — مجلد استبقاء كتيبات التدريب والدلائل المعمارية (Placeholder).
8. `events/` — مصفوفة كتالوج الأحداث الـ 9 الحاكمة.
9. `evidence/` — مجلد أدلة الإثبات والنزاهة الفنية السبعة للمرحلة صفر فاصلة خمسة.
10. `github/` — مجلد استبقاء روتينات وإدارة فروع ومستودعات GitHub (Placeholder).
11. `migration/` — مجلد استبقاء ترحيلات ومسودات قواعد البيانات قبل محاكاتها (Placeholder).
12. `permissions/` — نموذج التحكم بالأدوار والـ RBAC للوكلاء.
13. `protocols/` — البروتوكولات السبعة الحاكمة للمسارات البرمجية.
14. `reports/` — قوالب ومواصفات السجلات والتقارير الفنية.
15. `state/` — دورة حياة المهام الـ 12 والانتقالات الآمنة لحالات العمل.
16. `tests/` — مجلد استبقاء فحوصات الوحدة واختبارات الـ Smoke Run (Placeholder).
