# 📝 سجل التغييرات الرسمي للمنصة الفيدرالية (Changelog - V5)

يوثق هذا السجل كافة الإضافات، التعديلات، الإصلاحات، والتحصينات الأمنية المطبقة عبر إصدارات المنصة الفيدرالية الموحدة، توافقاً مع معايير وثيقة الإصدارات الدلالية المعتمدة.

---

## [v1.4.0] - 2026-06-29

### Added
* إضافة شجرة الصلاحيات والأدوار الموحدة (RBAC) لتقييد وتفويض المهام الوظيفية للموظفين بالفروع.
* إضافة سجل الرقابة والتدقيق التاريخي الموحد `audit_log` لدعم التحليلات والتوثيق للقطات البيانات قبل التعديل وبعده بالـ JSONB.

### Changed
* تفعيل سياسات أمان الصفوف والـ RLS الكلية و `FORCE RLS` بشكل إجباري على كافة الكائنات الاثني عشر لقاعدة البيانات.
* ربط مشغلات التحقق السريع بالخلفية لمنع تعديل أو حذف القيود المالية وفواتير المبيعات الصادرة بعد السداد والترحيل.

### Security
* عزل بيانات المنظمات والشركات الفيدرالية تماماً على مستوى محرك قاعدة البيانات لضمان حظر أي تداخل أو تسريب بين الكيانات المستأجرة.
* إغلاق وقفل جدول التدقيق والرقابة ليكون (Append-Only) وحظر أي عمليات حذف أو تعديل لصفوفه لضمان الحصانة القانونية.

---

## [v1.3.0] - 2026-06-28

### Added
* إضافة جداول جلسات محادثات الوكيل الذكي `ai_conversations` ورسائل سياق التاريخ الحواري `ai_messages`.
* إضافة جدول قواعد المعرفة والمستندات المرفوعة `ai_knowledge_base` لاستقبال اللوائح ووثائق الشركات.
* إضافة جدول الذواكر والبحث المتجهي `ai_memory` بالـ Vector Embeddings وفهارس الـ HNSW للبحث السيمانتك.

### Changed
* دمج محرك البحث السيمانتك مع الجداول التشغيلية للفروع لتوفير استشارات ذكية معزولة للموظفين.

### Security
* فرض سياسة حقن مرشح العزل للشركة آلياً قبل تفعيل حساب تشابه المتجهات اللغوية لمنع تسريب أسرار المعرفة التجارية.
* تفعيل بروتوكول حذف التبعيات التلقائي (Cascading Delete) لتصفير وحذف التضمينات المتجهية فور حذف المستند المعرفي.

---

## [v1.2.0] - 2026-06-27

### Added
* إضافة جداول الأرصدة والكميات الحية بالفروع `branch_inventory` وإصدار قيود إعادة الطلب الآمن.
* إضافة جداول معاملات النقل البيني والتسويات الجردية والكمية للمنتجات.
* إضافة فواتير المبيعات الصادرة وسلال الشراء المعلقة للكاشير وتوثيق الضرائب والإجماليات بدقة.
* إضافة شجرة الحسابات ودليل الأكواد المالية وجداول الدفتر المالي العام لقيود اليومية المزدوجة المتوازنة.
* إضافة جداول تتبع عهد الصناديق وورديات الكاشير اليومية وجداول تتبع ديون الذمم الملتزمة للشركاء.

### Changed
* ربط مسارات آلات الحالات لجميع المعاملات المالية والمخزنية لمنع الانتقال العشوائي والالتفاف التشغيلي للحالات.

### Fixed
* معالجة مشكلة تداخل حساب الضرائب والقيمة الإجمالية الصافية للفواتير وضبط دقة المجاميع دون انحراف عشري.

---

## [v1.1.0] - 2026-06-26

### Added
* إضافة جداول التصنيفات والأنواع السلعية والسلع والخدمات الرئيسية ومراقبة حدود الأسعار والضرائب المضافة.
* إضافة جدول الباركودات المتعددة والبديلة لربط السلعة الواحدة بعدة رموز تجارية.
* إضافة سجلات الموردين الخارجيين وعناوينهم وأرقام تراخيصهم الضريبية الرسمية بالدولة.
* إضافة حسابات العملاء ومستويات الولاء والائتمان الآجل لمتابعة التعهدات والجدارة الاستهلاكية.

### Changed
* فرض قيود التحقق لمنع إدخال أسعار شراء أو بيع سالبة أو نسب ضرائب تفوق 100% بالكتالوج البرمجي.

### Fixed
* تهيئة حساب العميل النقدي الافتراضي للفرع آلياً مع الترحيل لتسهيل حركات الكاشير المباشرة لليومية.

---

## [v1.0.0] - 2026-06-25

### Added
* تأسيس الهياكل التنظيمية العليا وجداول المنظمات والشركات العليا والترخيص الضريبي.
* إضافة جداول الفروع الجغرافية والمنشآت ونقاط البيع وربطها بالمنظمات الفيدرالية الأم.
* إضافة جداول الهوية الرقمية للموظفين والكوادر البشرية وحساباتهم وعضوياتهم المتعددة بالفروع.

### Changed
* تفعيل الامتدادات والملحقات التأسيسية لقاعدة البيانات وتشمل `uuid-ossp`, `pgcrypto`, `pg_trgm`, `pgvector`.

### Security
* إلزامية توليد المعرفات الفريدة والرموز باستخدام الـ UUID المعمى والمحمي بالكامل من التخمين العشوائي.
